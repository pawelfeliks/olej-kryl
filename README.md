# Krill Oil Landing Page

Static landing page prepared from the supplied Superba Krill, Revervia and specification materials.

## Structure

- `index.html` - complete page content in Polish
- `assets/css/main.css` - responsive styling
- `assets/js/main.js` - mobile menu handling
- `assets/images/` - graphics copied or rendered from the supplied materials

## Before production

1. Replace contact data in `index.html`.
2. Verify all health claims for the target market.
3. Replace product images with final licensed/approved assets.
4. Add privacy policy, terms and legal pages if the page will collect leads or sell products.
5. Confirm whether the Revervia algae section should stay on this page or be moved to a separate product page.

## Deployment

The package can be deployed as a simple static site on GitHub Pages, Azure Static Web Apps, Netlify, Vercel or any standard hosting.
